**Geomatics Engineering: Theodolites and Plane Table Surveying**
===========================================================

**Introduction**
---------------

Theodolites and plane table surveying are fundamental tools in geomatics engineering for measuring distances, angles, and coordinates of terrestrial objects. A theodolite is an optical instrument used to measure vertical and horizontal angles between two points on the Earth's surface, while a plane table is a graphical tool for recording observations and measurements.

**Core Concepts**
----------------

### Theodolites

A transit theodolite consists of several key components:

*   **Line of Collimation (LC):** The line connecting the object glass to the center of the telescope. It must be perpendicular to the horizontal axis at its intersection with the vertical axis.
*   **Axis of Altitude Level:** The axis that measures the vertical angle between the object and the instrument.
*   **Axis of Plate Level:** The axis that measures the horizontal angle between the object and the instrument.

### Plane Table Surveying

Plane table surveying involves recording observations and measurements on a graph paper with a set of concentric circles. The process includes:

1.  **Setting up the plane table:** Ensuring the horizontal surface is level and the vertical axis is plumb.
2.  **Measuring angles:** Using the theodolite to measure the horizontal and vertical angles between two points.
3.  **Recording measurements:** Tracing the measurements onto the graph paper.
