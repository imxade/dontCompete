**Theory of Computation: Context-Free Grammars and Pushdown Automata**
===========================================================

### Introduction
---------------

Context-free grammars (CFGs) and pushdown automata (PDAs) are fundamental concepts in theoretical computer science. A CFG is a set of production rules that generate strings, while a PDA is a type of automaton that recognizes languages generated by CFGs.

### Core Concepts
-----------------

#### Context-Free Grammars

A context-free grammar is defined as:

*   $G = (V_T, V_N, S, P)$, where:
    *   $V_T$ is the set of terminal symbols.
    *   $V_N$ is the set of non-terminal symbols.
    *   $S$ is the start symbol.
    *   $P$ is the set of production rules.

The production rules are of the form:

$$A \to \alpha$$

where $A$ is a non-terminal and $\alpha$ is a string of terminals and non-terminals. The start symbol $S$ is used to generate the strings in the language.

**Chomsky Normal Form (CNF)**:
---------------------------

In CNF, every production rule has one of two forms:

$$A \to BC$$
$$A \to a$$

where $a$ is a terminal and $B, C$ are non-terminals. This form simplifies the analysis of CFGs.

#### Pushdown Automata

A pushdown automaton is defined as:

*   $M = (Q, \Sigma, \Gamma, \delta, q_0, F)$, where:
    *   $Q$ is the set of states.
    *   $\Sigma$ is the input alphabet.
    *   $\Gamma$ is the stack alphabet.
    *   $\delta$ is the transition function.
    *   $q_0$ is the start state.
    *   $F$ is the set of final states.

The transition function $\delta$ defines the next state and the top of the stack:

$$\delta(q, a, \sigma) = (q', \gamma, z)$$

where $(q', \gamma, z)$ represents the new state, the new top of the stack, and the old top of the stack.

### Key Formulas/Theorems
-------------------------

The following are key formulas and theorems related to CFGs and PDAs:

*   **Pumping Lemma for Context-Free Languages**: If a language $L$ is context-free, then there exists an integer $p \ge 1$ such that every string in $L$ of length at least $p$ can be divided into five parts: $uvxyz$, where:
    *   $|vy| \le p$
    *   $|vx| \ge 0$
    *   $|xy| \ge 1$
    *   For all $i \ge 0$, if $uv^ixy^iz$ is in $L$, then so is $uv^{i+1}xy^{i+1}z$

*   **Myhill-Nerode Theorem**: A language is regular if and only if it has a finite index.

### Examples with Solutions
---------------------------

Here are some examples with solutions:

#### Example 1

Consider the CFG $G$ with production rules:

$$S \to aSa | bSb | ab$$

Determine whether the language generated by $G$ is regular or context-free.

**Solution**: The language generated by $G$ is context-free, as it can be proven using the Pumping Lemma.

#### Example 2

Consider the PDA $M$ with transition function:

$$\delta(q_0, a, \sigma) = (q_1, \gamma, z)$$

where $\sigma$ and $\gamma$ are stack symbols. Determine whether the language recognized by $M$ is regular or context-free.

**Solution**: The language recognized by $M$ is not regular, as it does not satisfy the Myhill-Nerode Theorem.

### Common Pitfalls
-----------------

*   **Confusing Language Properties**: Be cautious when analyzing the properties of a language. Make sure to use the correct methods and theorems.
*   **Ignoring Stack Operations**: When working with PDAs, do not ignore stack operations. They can significantly affect the recognition of languages.
